Three very useful functional programming tools are filter(), map() and reduce() filter(function,sequence) return a sequence with the items in the sequence that when test with the 'function' return 'true' 
>>> def func1(x):return x%2!=0 and x%3!=0

>>> filter(func1,range(20))
[1, 5, 7, 11, 13, 17, 19]
map(function,sequence) apply the 'function' to each element in the sequence and return a new sequence consisting every result 
>>> def func2(x):return x**2
>>> map(func2,range(2,5))
[4, 9, 16]
>>> map(func2,[2,3,4,5])
[4, 9, 16, 25]
>>> def func3(x,y):return x+y

>>> list1 =[1,2,3]
>>> list2 =[100,200,300]
the arguments number other than 'function' in map() must be the same as the argument number in the 'function' 
>>> map(func3,list1,list2)
[101, 202, 303]
reduce(function,sequence) apply the 'function' to first corresponding number of elements (2 in the example below) and take the result as one argument of the next run of 'function' until all elements are used 
>>> reduce(func3,list1)
6
>>> reduce(func3,list2)
600
